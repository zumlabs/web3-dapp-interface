# Web3 dApp Interface

A generic Web3 frontend interface built with Next.js for interacting with Ethereum-compatible smart contracts.

## Overview
This project serves as a reusable frontend foundation for Web3 applications, demonstrating wallet connectivity and smart contract interaction using modern frontend tooling.

## Tech Stack
- Next.js (App Router)
- TypeScript
- React
- Ethereum-compatible wallets (e.g., MetaMask)
